# Git Cmds

Git initialize
```
git init
```

Git ignore File
Make a file called .gitignore


Add to Stage
```
git add <filename>
git add -A
```

Status of the current repo
```
git status
```

Commit
```
git commit -m "Initial Commit"
```

Add a remote
```
git remote add origin <https_url>
```

Push to remote
```
git push origin master
```
